# 项目简介

## 核心功能
- 景点数据分析
- 景点信息管理
- 景点门票管理
- 景点类型管理
- 在线反馈
- 系统管理
- 个人中心
- 旅游资讯

## 适用场景
- 旅游规划
- 旅游景点推广
- 旅游爱好者查询
- 旅游行业数据分析

## 技术特点
- 使用 Django 作为后端框架
- 数据库采用 MySQL
- 前端采用 Vue.js
- 源码完整，附带详细文档
